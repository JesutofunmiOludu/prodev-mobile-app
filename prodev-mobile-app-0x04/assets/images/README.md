# Asset Files Required

Please download the required assets and place them in this directory:

- hero-icon.png (background image for landing screen)
- Logo.png (white logo for landing screen)
- logo-green.png (green logo for join/signin screens)
- google.png (Google logo for social login)
- facebook.png (Facebook logo for social login)

These assets are referenced in the `constants/index.ts` file.
